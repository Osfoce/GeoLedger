## 📖 About

GeoLedger is an open-source decentralized land registry platform built on the Stellar blockchain using Soroban smart contracts.

The project aims to modernize land registration by providing a transparent, secure, and tamper-resistant system for recording and managing property ownership.

Instead of relying solely on paper records or centralized databases, GeoLedger leverages blockchain technology to create verifiable and auditable ownership records.

---

## ✨ Features

- 🌍 Register land ownership on-chain
- 📍 Store parcel location information
- 📐 Record land size and property details
- 👤 Track property ownership
- 🔄 Secure ownership transfer
- 🛡 Immutable transaction history
- ⚡ Powered by Stellar Soroban smart contracts
- 📚 Open-source and community driven

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/JohnpiLabs/geoledger.git
```

Move into the project

```bash
cd geoledger
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Open

```
http://localhost:3000
```
